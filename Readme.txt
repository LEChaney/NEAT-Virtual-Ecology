Controls:
WASD to move around
Hold right click and move mouse to rotate camera
Spacebar to toggle between real-time and sped up time

Main project folders:
Scripts - All project code files
Scenes - All project scenes (environments)
Prefabs - Predefined gameobjects (Rabbits and Foxes)
CarExperiments - Race car scene and code file examples from UnityNEAT project.

Important config settings files:
SharpNeat/Network/DefaultActivationFunctionLibrary - Available activation functions specified in CreateLibraryNeat() function
SharpNeat/Genomes/Neat/NeatGenomeParameters - Mutation probabilities
